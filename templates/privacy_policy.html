Privacy policy of website antkeeping.info

Webserver logs

What personal data is collected?

When you access this website, the following information is stored in a server log file:

  • The page you visited
  • Date/Time of visit
  • The amount of data sent in Byte
  • Source/Link from which you accessed the page
  • The web browser you used
  • The operating system you used
  • Your IP-address

What is this data used for?

The data is used for two things:

  1. Protection and prevention of attacks against this website like DDoS, Brute force password attacks etc. Since it might be neccessary to conduct legal actions it is necessary to store the visitor's IP address.
  2. Creating anonymized web site statistics

How long is your personal data stored?

The personal data in the server log files is stored for a maximum of two weeks. After that your IP-address is anonymized.

The website statics are stored for an unlimited time. But since all the data used for it was anonymized previously it is not possible to make any connection to your person.

Cookies

This website only uses strictly necessary cookies, which do not require your consent:

  • Session cookie – keeps you logged in while browsing access-restricted pages. This cookie is only set when you log in and is deleted when you log out or close your browser session.
  • CSRF token – protects forms against Cross-Site Request Forgery attacks. This cookie is a standard security measure and contains no personal data.

No tracking, advertising, or analytics cookies are used on this website.

What rights do you have?

You have the right to ask what personal data is stored, request error correction and request deletion of personal data. To make these requests you have to send an email to: admin@antkeeping.info

In this email you must prove that you are eligible to receive this information. Otherwise your request cannot be processed.

Third party services

Cloudflare

This website is served through Cloudflare, which acts as a reverse proxy and CDN to protect the website against DDoS attacks and other threats. All traffic between your browser and this website passes through Cloudflare's infrastructure. As a result, Cloudflare processes connection data such as your IP address and HTTP request headers. Please refer to the Cloudflare Privacy Policy for details on how they handle this data.

No other third-party services are used. This website does not use any advertising networks, tracking services, or analytics platforms.
